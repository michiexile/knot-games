Interacting with the game
  • Drag knot tiles to the grid to place them
  • An illegally placed tile will return
  • Mouse Wheel will rotate the tile under the pointer
  • Repeated clicking will rotate the tile you are clicking on
  • Click once to select a tile
  • Use number keys 1, 2, 3... to select a selectable tile
  • Use number key 0, or click elsewhere, to de-select a tile
  • Keys [+] and [-] will rotate a selected tile
